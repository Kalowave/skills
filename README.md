# Kalowave skills

Reusable agent skills for Kalowave products.

## kaloclip

CLI wrapper + schema reference for the KaloClip Open API. See [SKILL.md](SKILL.md).

Quick start:

```bash
./scripts/kaloclip.sh set-key <your_api_key>
./scripts/kaloclip.sh help            # command list
./scripts/kaloclip.sh help <topic>    # per-endpoint schema
./scripts/kaloclip.sh flow <image-url> [title]   # end-to-end demo
```
