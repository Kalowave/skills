# Kalowave skills

Reusable agent skills for Kalowave products.

## kaloclip

CLI wrapper + schema reference for the KaloClip Open API. See [SKILL.md](SKILL.md).

Quick start:

```bash
./scripts/kaloclip.sh login               # device-flow: confirm in browser, key auto-saved
./scripts/kaloclip.sh help                # command list
./scripts/kaloclip.sh help <topic>        # per-endpoint schema + valid combos
```
